# @turnkey/spark

[![npm](https://img.shields.io/npm/v/@turnkey/spark?color=%234C48FF)](https://www.npmjs.com/package/@turnkey/spark)

[Turnkey](https://turnkey.com) Spark signer:

If you need a lower-level, fully typed HTTP client for interacting with Turnkey API, check out [`@turnkey/http`](https://www.npmjs.com/package/@turnkey/http).

API Docs: https://docs.turnkey.com/

## Getting started

```bash
$ npm install @turnkey/spark
```
