<H3Bordered text="Overview" />

Creates a new API key pair and returns the public key.

<ul>
  <li>This function generates a new API key pair and stores it in the underlying key store (IndexedDB).</li>
  <li>If an external key pair is provided, it will use that key pair for creation instead of generating a new one.</li>
  <li>If ``storeOverride`` is set to true, the generated or provided public key will be set as the override key in the API key stamper, making it the active key for subsequent signing operations.</li>
  <li>Ensures the API key stamper is initialized before proceeding.</li>
  <li>Handles both native CryptoKeyPair objects and raw key material.</li>
</ul>
