<H3Bordered text="Overview" />

Handles the login or sign-up flow.

<ul>
  <li>This function opens a modal with the AuthComponent, allowing the user to log in or sign up using any enabled authentication method (Passkey, Wallet, OTP, or OAuth).</li>
  <li>It automatically determines available authentication methods based on the current provider configuration and proxy settings.</li>
  <li>The modal-driven flow guides the user through the appropriate authentication steps, including social login if enabled.</li>
  <li>After successful authentication, the provider state is updated and all relevant session, user, and wallet data are refreshed.</li>
  <li>This function is typically used to trigger authentication from a UI button or navigation event.</li>
</ul>
