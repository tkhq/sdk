<H3Bordered text="Overview" />

Handles the signing of a message by displaying a modal for user interaction.

<ul>
  <li>This function opens a modal with the SignMessageModal component, prompting the user to review and approve the message signing request.</li>
  <li>Supports signing with any wallet account managed by Turnkey, including externally linked wallets.</li>
  <li>Allows for optional overrides of the encoding and hash function used for the payload, enabling advanced use cases or compatibility with specific blockchains.</li>
  <li>Optionally displays a subtext in the modal for additional context or instructions to the user.</li>
  <li>Returns a promise that resolves to a ``v1SignRawPayloadResult`` object containing the signed message, signature, and metadata.</li>
</ul>
