<p><strong>Package:</strong> <code>react-wallet-kit</code></p>

<H3Bordered text="Overview" />

Logs out the current client session.

<ul>
  <li>This function clears the specified session and removes any associated key pairs from storage.</li>
  <li>If a sessionKey is provided, it logs out from that session; otherwise, it logs out from the active session.</li>
  <li>Cleans up any api keys associated with the session.</li>
</ul>

<H3Bordered text="Parameters" />

<ParamField body="params" type='object' required={false} path="params" >
  <Expandable title="params details">
    <NestedParam parentKey="params" childKey="sessionKey" type='string' required={false}>
session key to specify which session to log out from (defaults to the active session).
</NestedParam>
  </Expandable>
</ParamField>

<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="returns" type="void" required={true}>
  A promise that resolves when the logout process is complete.
</ResponseField>
