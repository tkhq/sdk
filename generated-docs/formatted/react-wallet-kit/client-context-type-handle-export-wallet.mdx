<H3Bordered text="Overview" />

Handles the export wallet flow.

<ul>
  <li>This function opens a modal with the ExportComponent for exporting a wallet.</li>
  <li>Uses Turnkey's export iframe flow to securely export wallet material.</li>
  <li>The export process encrypts the exported bundle to a target public key, which is generated and managed inside the iframe for maximum security.</li>
  <li>A request is made to the Turnkey API to export the wallet, encrypted to the target public key.</li>
  <li>The resulting export bundle is injected into the iframe, where it is decrypted and displayed to the user.</li>
  <li>If a custom iframe URL is used, a target public key can be provided explicitly.</li>
  <li>Optionally allows specifying the stamper to use for the export (StamperType.Passkey, StamperType.ApiKey, or StamperType.Wallet) for granular authentication control.</li>
  <li>The modal-driven UI ensures the user is guided through the export process and can securely retrieve their exported material.</li>
</ul>
