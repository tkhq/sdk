<H3Bordered text="Overview" />

Handles the removal of a passkey (authenticator) for the user.

<ul>
  <li>This function opens a modal with the RemovePasskey component, allowing the user to confirm and remove a passkey authenticator from their account.</li>
  <li>It supports specifying the authenticator ID to remove, as well as optional modal title and subtitle for custom UI messaging.</li>
  <li>After successful removal, the user details state is refreshed to reflect the updated list of authenticators.</li>
  <li>Supports customizing the duration of the success page shown after removal.</li>
  <li>Allows specifying the stamper to use for the removal (StamperType.Passkey, StamperType.ApiKey, or StamperType.Wallet) for granular authentication control.</li>
  <li>Handles all error cases and throws a TurnkeyError with appropriate error codes.</li>
</ul>
