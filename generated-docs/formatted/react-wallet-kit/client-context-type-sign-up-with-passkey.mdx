<H3Bordered text="Overview" />

Signs up a user using a passkey, creating a new sub-organization and session.

<ul>
  <li>This function creates a new passkey authenticator and uses it to register a new sub-organization for the user.</li>
  <li>Handles both passkey creation and sub-organization creation in a single flow.</li>
  <li>Optionally accepts additional sub-organization parameters, a custom session key, a custom passkey display name, and a custom session expiration.</li>
  <li>Automatically generates a new API key pair for authentication and session management.</li>
  <li>Stores the resulting session token and manages cleanup of unused key pairs.</li>
</ul>

<H3Bordered text="Parameters" />

<ParamField body="params" type='object' required={false} path="params" >
  <Expandable title="params details">
    <NestedParam parentKey="params" childKey="createSubOrgParams" type='CreateSubOrgParams' required={false}>
parameters for creating a sub-organization (e.g., authenticators, user metadata).
</NestedParam>
    <NestedParam parentKey="params" childKey="expirationSeconds" type='string' required={false}>
session expiration time in seconds (defaults to the configured default).
</NestedParam>
    <NestedParam parentKey="params" childKey="passkeyDisplayName" type='string' required={false}>
display name for the passkey (defaults to a generated name based on the current timestamp).
</NestedParam>
    <NestedParam parentKey="params" childKey="sessionKey" type='string' required={false}>
session key to use for storing the session (defaults to the default session key).
</NestedParam>
  </Expandable>
</ParamField>

<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="returns" type="string" required={true}>
  A promise that resolves to a signed JWT session token for the new sub-organization.
</ResponseField>
