<p><strong>Package:</strong> <code>react-wallet-kit</code></p>

<H3Bordered text="Overview" />

Retrieves all sessions stored in persistent storage.

<ul>
  <li>This function fetches all session objects currently stored by the client, including those that are not active.</li>
  <li>Returns a record mapping each session key to its corresponding ``Session`` object.</li>
  <li>Useful for session management, auditing, or displaying all available sessions to the user.</li>
  <li>Automatically skips any session keys that do not have a valid session object.</li>
</ul>

<H3Bordered text="Parameters" />

<p>No parameters.</p>

<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="returns" type="undefined | Record&lt;string, Session&gt;" required={true}>
  A promise that resolves to a record of session keys and their corresponding `Session` objects, or `undefined` if no sessions exist.
</ResponseField>
