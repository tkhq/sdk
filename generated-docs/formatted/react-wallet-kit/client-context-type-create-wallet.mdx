<H3Bordered text="Overview" />

Creates a new wallet for sub-organization.

<ul>
  <li>This function creates a new wallet for the current sub-organization.</li>
  <li>If an organizationId is provided, the wallet will be created under that specific sub-organization; otherwise, it uses the current session's organizationId.</li>
  <li>If a list of address formats is provided, accounts will be created in the wallet based on those formats (starting from path index 0).</li>
  <li>If a list of account parameters is provided, those accounts will be created in the wallet.</li>
  <li>If no accounts or address formats are provided, default Ethereum and Solana accounts will be created.</li>
  <li>Optionally allows specifying the mnemonic length for the wallet seed phrase (defaults to 12).</li>
  <li>Optionally allows stamping the request with a specific stamper (StamperType.Passkey, StamperType.ApiKey, or StamperType.Wallet).</li>
</ul>
