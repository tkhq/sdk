<H3Bordered text="Overview" />

Handles the update user phone number flow.

<ul>
  <li>This function opens a modal with the UpdatePhoneNumber component for updating and verifying the user's phone number.</li>
  <li>If a phoneNumber is provided, it will directly send an OTP request to the user and display the OTP verification modal.</li>
  <li>Supports both manual entry and pre-filled phone numbers, as well as custom modal titles and subtitles.</li>
  <li>Uses the updatePhoneNumberContinue helper to manage the OTP flow, verification, and update logic.</li>
  <li>After successful verification and update, the user details state is refreshed and an optional success page can be shown.</li>
  <li>Supports customizing the duration of the success page after update.</li>
  <li>Throws a TurnkeyError if the client is not initialized, no active session is found, SMS OTP is not enabled, or if there is an error updating the phone number.</li>
</ul>
