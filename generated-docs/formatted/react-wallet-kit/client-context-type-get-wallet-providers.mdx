<p><strong>Package:</strong> <code>react-wallet-kit</code></p>

<H3Bordered text="Overview" />

Retrieves wallet providers from the initialized wallet manager.

<ul>
  <li>Optionally filters providers by the specified blockchain chain.</li>
  <li>Throws an error if the wallet manager is not initialized.</li>
</ul>

<H3Bordered text="Parameters" />

<ParamField body="chain" type='Chain' required={false} >
  optional blockchain chain to filter the returned providers.
</ParamField>

<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="returns" type="WalletProvider[]" required={true}>
  A promise that resolves to an array of wallet providers.
</ResponseField>
