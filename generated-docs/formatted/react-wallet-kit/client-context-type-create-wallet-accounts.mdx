<H3Bordered text="Overview" />

Creates new accounts in the specified wallet.

<ul>
  <li>This function creates new wallet accounts based on the provided account parameters or address formats.</li>
  <li>If a walletId is provided, it creates the accounts in that specific wallet; otherwise, it uses the current session's wallet.</li>
  <li>If a list of address formats is provided, it will create accounts in the wallet based on those formats, automatically determining the next available path indexes to avoid duplicates with existing accounts.</li>
  <li>If account parameters are provided, they are used directly for account creation.</li>
  <li>Automatically queries existing wallet accounts to prevent duplicate account creation for the same address format and path.</li>
  <li>Supports stamping the request with a specific stamper (StamperType.Passkey, StamperType.ApiKey, or StamperType.Wallet).</li>
</ul>
