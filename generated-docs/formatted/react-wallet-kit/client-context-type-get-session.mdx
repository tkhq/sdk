<p><strong>Package:</strong> <code>react-wallet-kit</code></p>

<H3Bordered text="Overview" />

Retrieves the session associated with the specified session key, or the active session by default.

<ul>
  <li>This function retrieves the session object from storage, using the provided session key or, if not specified, the current active session key.</li>
  <li>If no session key is provided and there is no active session, it returns undefined.</li>
  <li>Returns the session details, including public key, organization ID, user ID, and expiration.</li>
</ul>

<H3Bordered text="Parameters" />

<ParamField body="params" type='GetSessionParams' required={false} >
  <Expandable title="params details">
    <NestedParam parentKey="params" childKey="sessionKey" type='string' required={false}>
session key to retrieve a specific session (defaults to the current active session key).
</NestedParam>
  </Expandable>
</ParamField>

<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="returns" type="undefined | Session" required={true}>
  A promise that resolves to a `Session` object containing the session details, or undefined if not found.
</ResponseField>
