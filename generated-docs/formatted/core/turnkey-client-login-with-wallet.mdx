<H3Bordered text="Overview" />

Logs in a user using the specified wallet provider.

<ul>
  <li>This function logs in a user by authenticating with the provided wallet provider via a wallet-based signature.</li>
  <li>If a public key is not provided, a new one will be generated for authentication.</li>
  <li>Optionally accepts a custom session key and session expiration time.</li>
  <li>Stores the resulting session token under the specified session key, or the default session key if not provided.</li>
  <li>Throws an error if a public key cannot be found or generated, or if the login process fails.</li>
</ul>
