<H3Bordered text="Overview" />

Removes passkeys (authenticator) from the user.

<ul>
  <li>This function removes passkeys (WebAuthn/FIDO2 authenticators) from the user's account.</li>
  <li>If a userId is provided, it removes the passkeys for that specific user; otherwise, it uses the current session's userId.</li>
  <li>Automatically ensures an active session exists before making the request.</li>
  <li>Optionally allows stamping the request with a specific stamper (StamperType.Passkey, StamperType.ApiKey, or StamperType.Wallet).</li>
  <li>Returns an array of remaining authenticator IDs for the user after removal.</li>
</ul>
