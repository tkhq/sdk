<H3Bordered text="Overview" />

Fetches the user details for the current session or a specified user.

<ul>
  <li>Retrieves user details from the Turnkey API using the provided userId and organizationId, or defaults to those from the active session.</li>
  <li>If no userId is provided, the userId from the current session is used.</li>
  <li>If no organizationId is provided, the organizationId from the current session is used.</li>
  <li>Optionally allows stamping the request with a specific stamper (StamperType.Passkey, StamperType.ApiKey, or StamperType.Wallet).</li>
  <li>Ensures that an active session exists before making the request.</li>
</ul>
