<H3Bordered text="Overview" />

Signs up a user using an OTP verification token.

<ul>
  <li>This function signs up a user using the verification token received after OTP verification (from email or SMS).</li>
  <li>Creates a new sub-organization for the user with the provided parameters and associates the contact (email or phone) with the sub-organization.</li>
  <li>Automatically generates a new API key pair for authentication and session management.</li>
  <li>Stores the resulting session token under the specified session key, or the default session key if not provided.</li>
  <li>Handles both email and SMS OTP types, and supports additional sub-organization creation parameters.</li>
</ul>
