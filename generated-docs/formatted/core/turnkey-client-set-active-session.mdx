<H3Bordered text="Overview" />

Sets the active session to the specified session key.

<ul>
  <li>This function updates the ``activeSessionKey`` in persistent storage to the specified session key.</li>
  <li>Ensures that subsequent operations use the session associated with this key as the active session.</li>
  <li>Does not validate whether the session key exists or is valid; it simply updates the pointer.</li>
  <li>Useful for switching between multiple stored sessions or restoring a previous session context.</li>
</ul>
