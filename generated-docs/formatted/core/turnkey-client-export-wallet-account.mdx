<H3Bordered text="Overview" />

Exports a wallet account as an encrypted bundle.

<ul>
  <li>This function exports the specified wallet account as an encrypted bundle, suitable for backup or transfer.</li>
  <li>The exported bundle contains the wallet account's key material, encrypted to the provided target public key.</li>
  <li>If a targetPublicKey is provided, the bundle will be encrypted to that public key; otherwise, an error will be thrown.</li>
  <li>If an organizationId is provided, the wallet account will be exported under that sub-organization; otherwise, the current session's organizationId is used.</li>
  <li>Optionally allows stamping the request with a specific stamper (StamperType.Passkey, StamperType.ApiKey, or StamperType.Wallet).</li>
</ul>
