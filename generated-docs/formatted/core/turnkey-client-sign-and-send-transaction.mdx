<H3Bordered text="Overview" />

Signs and broadcasts a transaction using the specified wallet account.

<ul>
  <li>For **connected wallets**:</li>
  <li>Calls the walletâ€™s native ``signAndSendTransaction`` method.</li>
  <li>Does **not** require an ``rpcUrl``.</li>
</ul>

<ul>
  <li>For **embedded wallets**:</li>
  <li>Signs the transaction using the Turnkey API.</li>
  <li>Requires an ``rpcUrl`` to broadcast the transaction.</li>
  <li>Broadcasts the transaction using a JSON-RPC client.</li>
</ul>
