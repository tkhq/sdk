<H3Bordered text="Overview" />

Signs up a user using a passkey, creating a new sub-organization and session.

<ul>
  <li>This function creates a new passkey authenticator and uses it to register a new sub-organization for the user.</li>
  <li>Handles both passkey creation and sub-organization creation in a single flow.</li>
  <li>Optionally accepts additional sub-organization parameters, a custom session key, a custom passkey display name, and a custom session expiration.</li>
  <li>Automatically generates a new API key pair for authentication and session management.</li>
  <li>Stores the resulting session token and manages cleanup of unused key pairs.</li>
</ul>
