<H3Bordered text="Overview" />

Verifies the OTP code sent to the user.

<ul>
  <li>This function verifies the OTP code entered by the user against the OTP sent to their contact information (email or phone) using the auth proxy.</li>
  <li>If verification is successful, it returns the sub-organization ID associated with the contact (if it exists) and a verification token.</li>
  <li>The verification token can be used for subsequent login or sign-up flows.</li>
  <li>Handles both email and SMS OTP types.</li>
</ul>
